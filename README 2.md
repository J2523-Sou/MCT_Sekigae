# 席替えアプリ

1台の端末だけで席替えを完結できるアプリを開発中です。

## 構成

- `ver_server/` — 同じ Wi-Fi 上の複数端末から利用する、従来のサーバー版です。実装と利用手順は [ver_server/README.md](ver_server/README.md) を参照してください。

端末完結版の実装は、このリポジトリのルートに追加する予定です。

## Git 管理しないファイル

投票データを保存する SQLite データベース、Python のキャッシュ、ローカル設定ファイルは `.gitignore` で除外しています。実際の投票データや管理者パスワードをリポジトリへコミットしないでください。
