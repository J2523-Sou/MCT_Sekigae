# 席替えアプリ

同じ Wi-Fi 上の複数端末から使うネットワーク版を `ver_server/` で管理しています。

利用方法と実装の詳細は [ver_server/README.md](ver_server/README.md) を参照してください。

## 構成

- `ver_server/` — 席替え投票アプリ本体。起動スクリプト、サーバー、画面ファイルを含みます。

## Git 管理しないファイル

投票データを保存する SQLite データベース、Python のキャッシュ、ローカル設定ファイルは `.gitignore` で除外しています。投票データや管理者パスワードをリポジトリへコミットしないでください。
